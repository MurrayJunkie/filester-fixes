# Filester-Fixes

> A userscript that adds small quality-of-life improvements to Filester.  
> I have some other problems with the website that I might have an attempt at fixing but they are not as easy as the currently offered solutions.  
> Tested on Firefox. It should work on any browser that supports Tampermonkey.

---

## Currently Offered Features

- Brings back the good old links to the album items. So you can open them in a new url or in the same page. HOWEVER you prefer to.
- Inserts a referrer link to downloads to allow IDM to know the origin of the download for example.


---

## Usage

Are finding the design choices of the website annoying?  
This attempts to fix some of those design choices to fit my preferances.  

---

## Install

### 1. Install Tampermonkey

Install it from your browser’s extension store.

### 2. Install the script

- Open the [user.js](./user.js) file.  
- Click the **Raw** button.  
- Tampermonkey should prompt you to install it.  
- If it doesn’t, create a new userscript manually and paste the contents of `user.js`.

### 3. Done

Website should allow you to navigate it a bit easier now.
